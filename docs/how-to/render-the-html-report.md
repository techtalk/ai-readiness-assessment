# Render the HTML report

The assessment can produce a single-file, shareable HTML version of the
report — useful for handing to a lead or stakeholder who won't open the
markdown.

## Ask for it

After an assessment has run:

> Render this assessment as a shareable HTML page.

The tool produces the HTML **only when asked** — it never writes it
unprompted.

## What you get

A single, print-friendly page (no emoji, no animations) with:

- A header showing the project, date, the **Habitat Maturity Level**
  badge and the **cognitive level** badge (colour-coded L0–L5), and the
  one-line rationale.
- A **Habitat Maturity Profile** block — all fourteen dimensions as
  small bars labelled with the model's verb at the placed level;
  behavioural dimensions placed without direct evidence carry an
  "(inferred)" tag.
- Three **discipline cards** with 0–5 strength indicators.
- An **Operational Axes** strip (the four headline dimensions) and a
  **Habitat Build Gap** badge showing the signed gap and its regime.
- Strengths and gaps side by side, numbered recommendations, the reading
  path with a clear Leanpub button, and the single TechTalk call to
  action.

## Tips

- It's a self-contained file — open it in a browser or print to PDF to
  circulate.
- Re-render after each assessment; keep the markdown under
  `assessments/` as the source of record.
